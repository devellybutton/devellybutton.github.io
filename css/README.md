# CSS 파일 구조
## /base
- `reset.css`: 리셋 및 기본 스타일
- `typography.css`: 폰트 및 텍스트 관련 스타일
- `variables.css`: 변수 정의 (색상, 크기 등)

## /components
- `buttons.css`: 버튼 스타일
- `cards.css`: 카드 컴포넌트
- `forms.css`: 폼 요소 스타일
- `top-button.css`: 맨 위로가기 버튼 스타일

## /cursor
- `cursor.css`: 커서 스타일

## /layout
- `header.css`: 헤더 스타일
- `footer.css`: 푸터 스타일

## /responsive
- `responsive.css`: 반응형 모음

## /sections
- `hero.css`: 히어로 섹션
- `my-name.css`: 이름 애니메이션 부분
- `about.css`: 소개 섹션
- `portfolio.css`: 포트폴리오 섹션
- `education.css`: 교육 섹션
- `career.css`: 경력 섹션
- `license.css`: 자격증 섹션
- `contact.css`: 연락처 섹션

## 최상위 파일
- `main.css`: 모든 CSS 파일을 불러오는 메인 파일
